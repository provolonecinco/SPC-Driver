# Unnamed SPC700 Audio Driver
A Super NES hobby project with **no deadline**.

## Building
Builds with **GNU Make**. Make sure you have CC65 installed and **CA65/LD65** are added to your PATH. 
```
make clean
make
```
Produces `output/sound.sfc`. Enjoy!
